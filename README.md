# IoT 기기 연동 및 웹, 앱 식물 커뮤니티 풀스택 개발 (Python, Java/Spring, React·React Native)

## 📖 개요
- IoT 기기 자동제어 알고리즘 개발 (Python, Raspberry Pi)
- 백엔드: Java/Spring + MariaDB로 API 서버 구축
- 프론트엔드: React + React Native로 웹·앱 연동 구현
